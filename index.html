历史文本情感与语义演变分析 (Social Science Quantitative Analysis)

基于 Python (Jieba & Pandas) 的文本分析项目，用于探索如何使用基于词典的方法分析历史文本中的叙事模式演变。
通过构建加权语义词典，量化分析了过去60年间某国特定公开纪念演讲中的叙事变化，并使用 Matplotlib 绘制了双面板趋势图。

Semantic Evolution in Historical Texts

A text mining project powered by Python (Jieba & Pandas). This project explores how narrative modes evolve in historical texts using a dictionary-based approach.
Quantified keyword frequency shifts in public commemorative speeches over 60 years using weighted dictionaries, visualized via Matplotlib dual-panel plotting.
